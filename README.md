Learning Vulkan
=========
Repository of me learning Vulkan in modern C++.
Tutorial I use: <https://vulkan-tutorial.com/Introduction>

Dependencies
---------------
Arch Dependencies:

- vulkan-headers
- vulkan-validation-layers
- vulkan-tools
- glfw-x11
- glfw-doc (optional)
- glm
- shaderc

How To Build
--------------
Build:
```
$ make
```
Build and run:
```
$ make test
```
